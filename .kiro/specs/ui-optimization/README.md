## 🎨 设计原则

1. **渐进增强** - 从基础功能开始，逐步添加增强
2. **移动优先** - 优先考虑移动端体验
3. **一致性** - 统一的颜色、间距、字体系统
4. **可访问性** - 符合WCAG 2.1 AA标准
5. **性能优先** - 优化加载和渲染性能

## 📈 关键指标

### 性能指标
- ✅ 首次渲染 < 1秒
- ✅ 动画帧率 ≥ 60fps
- ✅ 移动端可用性 > 90分

### 可访问性指标
- ✅ WCAG 2.1 AA级别
- ✅ 键盘可访问性 100%
- ✅ 屏幕阅读器兼容

### 浏览器兼容性
- ✅ Chrome 90+
- ✅ Firefox 88+
- ✅ Safari 14+
- ✅ Edge 90+

## 🎨 核心改进预览

### 1. Header优化

**优化前：**
```
┌─────────────────────────────────┐
│ [卡片] BIP39/SLIP39 | 在线 | 下载│
└─────────────────────────────────┘
```

**优化后：**
```

│                    [下载离线版]  │

  ↑ 渐变紫色背景，圆角阴影
```

### 2. 助记词显示优化

**优化前：**
```
[1] abandon  [2] ability  [3] able
[4] about    [5] above    [6] absent
...
```

**优化后：**
```
┌────────┐ ┌────────┐ ┌────────┐ ┌────────┐
│ ① abandon│ │ ② ability│ │ ③ able  │ │ ④ about │
└────────┘ └────────┘ └────────┘ └────────┘

```

### 3. 按钮组优化

**优化前：**
```
[生成] [粘贴] [转换] [清除]
```

**优化后：**
```
主要操作：[🔄 生成] [📋 粘贴]
次要操作：[🔀 转换] [🗑️ 清除]
  ↑ 分组显示，带图标
```

## 🛠️ 技术栈

- **框架**: Vue 3 (Composition API)
- **UI库**: Element Plus 2.13.1
- **样式**: SCSS + CSS Grid/Flexbox
- **构建**: Vite 7.0.6
- **响应式**: Media Queries

## 📝 样式变量速查

### 颜色
```scss
$primary: #667eea;    // 主色
$success: #67c23a;    // 成功
$warning: #e6a23c;    // 警告
$danger: #f56c6c;     // 危险
```

### 间距
```scss
$spacing-xs: 8px;     // 超小
$spacing-sm: 12px;    // 小
$spacing-md: 16px;    // 中
$spacing-lg: 24px;    // 大
$spacing-xl: 32px;    // 超大
```

### 字体
```scss
$font-size-h1: 24px;  // 主标题
$font-size-base: 14px;// 正文
$font-size-small: 12px;// 辅助
```
